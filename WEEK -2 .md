# Week 2 – Document-to-Quiz Chatbot

## Goal  
Build the core logic that transforms uploaded study documents into interactive quiz questions using an LLM.

## Achievements  

### 📂 PDF Processing Capability  
- Implemented automatic PDF upload and text extraction.  
- System now ingests study materials directly and prepares them for AI processing.  

### 🤖 Document-to-Quiz Chatbot  
- Connected extracted text to an LLM (tested with GPT‑3.5 / local MPT‑7B).  
- Designed prompts to generate **intelligent quiz questions** from educational content.  
- Built chatbot logic to deliver questions interactively to the user.  

### 🧩 Text Processing & Question Generation  
- Implemented a pipeline that cleans and structures raw text.  
- Generated multiple‑choice and short‑answer questions.  
- Added **difficulty levels** (easy, medium, hard) to diversify practice.  

## Outcome  
By the end of Week 2, the system can:  
- Upload study PDFs  
- Extract text automatically  
- Generate quiz questions from documents  
- Deliver questions with varying difficulty levels via a chatbot interface
