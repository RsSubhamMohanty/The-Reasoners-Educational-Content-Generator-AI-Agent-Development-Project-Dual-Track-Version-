## 🚀 Week 5: Building the User Interface with Streamlit

In this week, we focused on developing the **front-end interface** of our AI Study Assistant using **Streamlit**.

Streamlit allows us to convert our AI logic into an interactive web application where users can easily upload documents and generate study materials.

---

## 🖥️ Interface Design

The application interface is designed to be simple and user-friendly:

```
Upload PDF
────────────────

[ Generate Quiz ]

[ Generate Flashcards ]

[ Generate Summary ]
```

---

## ⚙️ Implementation Details

* Used **Streamlit** to create a web-based interface
* Added a **file uploader** to allow users to upload PDF documents
* Designed interactive buttons for:

  * Quiz generation
  * Flashcard creation
  * Summary generation

---

## 🔄 Working Flow

1. The user uploads a PDF file
2. The system reads and extracts text from the document
3. Based on user selection:

   * Quiz questions are generated
   * Flashcards are created
   * Summary is produced

---

## 🎯 Outcome of Week 5

* Successfully built a **functional UI** for the AI system
* Enabled **user interaction with backend AI logic**
* Created a foundation for adding advanced features in upcoming weeks

---

## 💡 Key Learning

* Understanding how to use Streamlit for rapid UI development
* Connecting frontend components with backend AI processing
* Handling user inputs and displaying dynamic outputs
